Here is the some steps to run this application.

Clone application from git hub link
Open the application in Visual studio code
Before run the application you have to add all required dependancies from NPM by using the command - NPM i
Once all npm libraries downloaded run the angular application using command - ng serve
Once the code compiled open the browser and hit the link - http://localhost:4200/ OR respective Port number
now it will navigates to the manager login page
First create manager account by signin page
Once manager account is created you can log in with credentials
The you can access the application.
